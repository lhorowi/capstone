This shows the structure of data returned by the API after processing a syllabus.

Response Structure

success (boolean)
true if successful
data (object)
Contains the extracted syllabus information with the structure below:

üìö data.courseInfo

Course basic information

courseName (string)
Full course name
courseCode (string)
Course code/number (e.g., "REL100")
instructor (string)
Instructor full name
email (string)
Instructor email address
officeHours (string)
Office hours information
officeLocation (string)
Office location/room

üìÖ data.datesAndDeadlines

Array of all important dates and deadlines

Each item contains:
date (string)
Exact date as written (e.g., "September 30" or "October 16")
title (string)
Name of the assignment, exam, or event
type (string)
Type: "assignment", "exam", "quiz", "deadline", "holiday", "break"
description (string)
Detailed description
worth (string)
Points or percentage (if mentioned)
link (string)
URL or link if attached in the document

Array of all assignments

Each item contains:
name (string)
Assignment name
description (string)
Full description of what's required
dueDate (string)
Due date (use format from document)
weight (string)
Percentage of final grade
requirements (string)
Specific requirements and formatting

üéØ data.gradingBreakdown

Array of grading categories

Each item contains:
category (string)
Category name (e.g., "Participation")
weight (string)
Percentage or points (e.g., "10%")
description (string)
What's included in this category

üìã data.policies

Array of course policies

Each item contains:
type (string)
Policy type: "attendance", "grading", "late work", "academic integrity", "participation"
title (string)
Policy heading
description (string)
Complete policy text

üìÜ data.schedule

Array of weekly schedule

Each item contains:
week (string)
Week number (e.g., "Week 1")
date (string)
Date or date range
topic (string)
Topic/theme
readings (string)
Required readings
assignments (string)
Assignments due this week

üë§ data.instructorContact

Instructor contact information

name (string)
Full name
email (string)
Email address
officeHours (string)
Office hours
office (string)
Office location

üìñ data.requiredMaterials

Array of required course materials

Each item contains:
type (string)
Material type: "textbook", "article", "website"
title (string)
Material title
author (string)
Author name
details (string)
Additional details (edition, ISBN, etc.)
